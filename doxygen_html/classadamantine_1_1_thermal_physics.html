Detailed Description

template<int dim, int fe_degree, typename NumberType, typename QuadratureType>
class adamantine::ThermalPhysics< dim, fe_degree, NumberType, QuadratureType >

This class takes care of building the linear operator and the right-hand-side. Also used to evolve the system in time.
